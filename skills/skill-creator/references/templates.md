# スキルテンプレート集

## テンプレート1: 対話型スキル

ユーザーと会話しながら成果物を作るスキル。

```markdown
---
name: <skill-name>
description: [何をするか]。[いつ使うか]。「[呼び出しフレーズ]」で使用。
---

# [スキル名]

[1行の説明]

## 手順

### Step 1: 確認

ユーザーに以下を確認：
- [確認事項1]
- [確認事項2]

### Step 2: 作成

[作成手順]

### Step 3: 出力

[出力フォーマット]

## 使用例

```
[呼び出し例]
```
```

description: [何をするか]。[いつ使うか]。「[呼び出しフレーズ]」で使用。
---

# [スキル名]

## 手順（必ずこの順番で実行）

### Step 1: データ取得

```bash
[コマンド]
```

### Step 2: 加工・選別

[選別ルール]

### Step 3: フォーマットして返信

```
[出力テンプレート]
```

## 重要

- [制約事項]
```

[整理の観点]

### Step 2: 分析

以下の形で提示：

- 概要:
- 理由:

- 概要:
- 理由:

```

```

---

## descriptionの書き方

### 必須要素
1. **何をするか** - スキルの主機能
2. **いつ使うか** - 適用場面
3. **呼び出しフレーズ** - 「〇〇して」

### 適切な長さ
50-150文字程度。短すぎても長すぎてもダメ。
