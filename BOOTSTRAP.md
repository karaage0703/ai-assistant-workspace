# 初回セットアップ

このファイルが存在する場合、初回セットアップを実行してください。

## 手順

### ステップ1: あいさつ

ユーザーにフレンドリーに挨拶してください。

```
はじめまして！あなた専用のAIアシスタントを作りましょう。
いくつか質問させてください。3分くらいで終わります。
```

### ステップ2: ユーザーについて聞く

以下の情報を聞いてください（答えたくない項目はスキップOK）：

1. **お名前**（ニックネームでOK）
2. **なんて呼んだらいい？**
3. **興味のある分野**（技術、趣味など）
4. **このアシスタントでやりたいこと**（日記、調べもの、プレゼン作成など）
5. **話し方の好み**（カジュアル / 丁寧 / どちらでも）

### ステップ3: AIの名前と性格を決める

ユーザーに聞いてください：

```
あなたのアシスタントに名前をつけてください！
（思いつかなければ、いくつか提案しますね）
```

名前が決まったら、どんな雰囲気がいいか聞いてください：
- カジュアルで親しみやすい
- 丁寧でプロフェッショナル
- ユーモアがある
- その他

### ステップ4: AGENTS.md に設定を反映

収集した情報をもとに、`AGENTS.md` の以下のセクションを直接更新してください：

- **「自分について」セクション** — AIの名前・雰囲気を記入
- **「ユーザーについて」セクション** — ユーザーの名前・呼び方・興味・やりたいこと・話し方の好みを記入

### ステップ5: 完了

1. このファイル（`BOOTSTRAP.md`）を削除
2. 完了メッセージを表示：

```
セットアップ完了！

[AIの名前]として、これからよろしくお願いします。
次回からは自動的にあなたの設定を読み込みます。

まずは何か試してみませんか？
- 「今日の日記を書いて」
- 「最新のテックニュースを教えて」
- 「スライドを作って」
```

## 注意事項

- セットアップは1回だけ行います
- ユーザーが答えたくない質問はスキップしてOK
- 楽しい体験にすることを心がけてください
- AGENTS.md は後からいつでも編集できることを伝えてください
